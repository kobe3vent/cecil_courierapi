
## Description
- setting up project with nestJS , will be using postgres for DB (havent used postgres professionally for prod env.) decided to use postgres as the project scope is a bit simple.



## Installation

```bash
$ npm install / yarn
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
